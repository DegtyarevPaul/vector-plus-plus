Вам заданы два простых числа х и у (х больше у). Вы можете выбрать любое
простое число м и вычесть его n раз из х. (n- тоже простое число). Можно
ли сделать х равным у?

Простым числом называется такое число, которое [имеет ровно два
положительных делителя]{.underline}: 1 и само себя.

*Вам необходимо дать ответ на t независимых тестов.\
***Входные данные**\
\
Первая строка входных данных содержит целое число t-количество тестов.\
\
Следующие t строк описывают тесты. Каждая строка содержит два целых
числа х и у.

**Выходные данные**\
\
Ответ на вопрос в виде слова: YES , если можно выполнить условие и: NO ,
если нельзя

**Ограничения**

$$\ 1\  \leq \ t\  \leq \ 1000\ \ \ \ \ 1\  \leq \ y\  < \ x\  \leq \ 10^{18}$$

**Пример**\
\
Ввод: 

3

11 10

23 20

49 14

Вывод:\
NO

NO

YES
